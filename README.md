# Aula Clara

Aula Clara es un sistema web para la gestión de clases particulares, seguimiento de alumnos y organización de materiales educativos.

## Objetivo

Construir una herramienta simple y útil para que profes particulares puedan organizar alumnos, materias, clases, ejercicios, materiales y seguimiento docente.
